# Greek Materials — Papakosta Tutoring

Interactive learning resources for Modern Greek, organized by CEFR level.

All materials are built by **Papakosta Tutoring** for use with private students and as part of a broader curriculum aligned to the Πιστοποίηση Ελληνομάθειας certification framework.

## Levels

| Level | Folder | Description |
|-------|--------|-------------|
| **A1** | `a1/` | Absolute beginner — alphabet, greetings, classroom vocabulary, colors |
| **A2** | `a2/` | Elementary — daily routines, family, food, simple past |
| **B1** | `b1/` | Intermediate — narration, opinion, basic argumentation |
| **B2** | `b2/` | Upper-intermediate — complex texts, written expression |
| **Ellinomathia** | `ellinomathia/` | Targeted exam preparation for the Greek language certification |

## How to use

Each level folder contains standalone interactive HTML files.

- **Online:** open the GitHub Pages link. For example, the A1 escape room is at  
  `https://papakostatutoring-star.github.io/greek-materials/a1/escape-magiki-taxi.html`
- **Offline:** download the HTML file and double-click it. Runs in any modern browser.

## Featured

### 🦉 Η Μαγική Τάξη (A1 Escape Room)
